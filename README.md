# Studyapp
Simple web app created to help students organize their school and personal life with the help of several tools. A music player, to-do list and scedule provide a basic framework for student productivity.

## Features
##### Universal
- Mobile adaptability and PC friendliness
- Connected navigation across pages
- Persistent storage across pages
- Dark and light mode with accent colors that can be adjusted
- quick view of todo items and calender
- simple note interface

 ##### Home page
-  Welcome message introducing users to the website
-  Functional Music player and playlist with ability to upload local files
-  To do reminders
-  Upcoming timetable events for current day

##### To do page
- Functional to-do list with ability to add, remove and edit items
- connection bettween todo item and time table

##### Timetable page
- Shows timetable
- show todo items if they were given a date
  
##### About page
- Details about app and its version
- Creator info 
- Shifting display for creators
